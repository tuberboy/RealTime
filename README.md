# RealTime
Real Time (live) - Date and Time using c and also in php, it's look like javascript loop date time

* only works on terminal or on any command line interface

# Build C
```
git clone https://github.com/TuberBoy/RealTime.git
cd RealTime
cc -o dt main.c
./dt
```
# Run PHP
```
git clone https://github.com/TuberBoy/RealTime.git
cd RealTime
php main.php
```
